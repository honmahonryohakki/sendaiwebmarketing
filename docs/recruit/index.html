働きやすさ・サポート

柔軟な働き方

直行直帰やオンライン商談を取り入れ、フレックス制で成果に集中できる環境を整えています。家庭や副業との両立も相談できます。

営業・マーケ連携

広告運用チームやデータアナリストと連携し、提案精度を高める体制を整備。社内ナレッジ共有で商談の質向上をサポートします。

スキルアップ支援

外部研修や資格取得を補助し、マーケティング・AI活用の最新知識を習得できます。営業資料テンプレートや成功事例も共有しています。

選考フロー

  1. エントリー：フォームまたはメールで履歴書・職務経歴書をお送りください。
  2. カジュアル面談：これまでのご経験や志向性を伺い、会社概要をご説明します。
  3. 提案課題：簡単な提案書作成または商談ロールプレイを実施します。
  4. 最終面談：代表・営業リーダーとの面談で、入社後の役割や条件を擦り合わせます。
  5. オファー：条件提示と入社時期の調整を行います。

スケジュールや選考ステップはご状況に応じて柔軟に調整可能です。お気軽にご相談ください。
